# TaxRateStatus

Status of the tax rate in the accounting platform.  
- `Active` - An active tax rate in use by a company.  
- `Archived` - A tax rate that has been archived or is inactive in the accounting platform.  


## Values

| Name       | Value      |
| ---------- | ---------- |
| `ACTIVE`   | Active     |
| `ARCHIVED` | Archived   |
